\# 2D Ball Rigid Physics Engine (SDL3)



A simple 2D physics simulation using SDL3 to demonstrate ball movement, collisions, and basic physics.



\## Features



Ball spawning, gravity toggle, collision detection, wall bouncing, damping, speed control, and mouse drag shooting.



\## Controls



Left Click: spawn ball

Right Click + Drag: shoot ball

Space: spawn center ball

G: toggle gravity

R: remove last ball

C: clear all balls

Up/Down: change speed

D/F: change damping



\## Tech



C++, SDL3, SDL\_image, SDL\_ttf



\## About SDL



SDL (Simple DirectMedia Layer) is a library used to handle graphics, input, audio, and window management for games and multimedia applications.



Official site: https://www.libsdl.org/
